Como instalo Visual Studio Code?




Para empezar a programar HTML5 deberas tener una terminal donde programarlo, y aunque se podria programar en un simple bloc de notas eso no es nada practico, porque no tendrias las mísmas facilidades que en una terminal dedicada.

Para comenzar con la instalacion de Visual Studio Code deberas hacer click en este link para ir a la pagina oficial de visual studio code, descargar la version que necesites y hacer una instalacion normal.

Una vez instalado y abierto ya se podria usar, pero le añadiremos una cosa para que sea mucho más facil de usar, lo llamado pluguins. Pero antes de eso deberemos crear una carpeta en un sito accesible del ordenador, ya que accederemos frecuentemente en este caso yo la cree en el escritorio, una vez creada volvemos a Visual Studio Code y hacemos click en seleccionar carpeta, y seleccionamos esa.

Ahora si podemos instalar ya el pluguin que necesitaremos para facilitar la programacion, el cual sera el live server. Para instalarlo debereis ir a la barra lateral de la izquierda y seleccionar extensiones la cual estara simbolizada con unos cuadraditos. Una vez ahi escribiras "Live Server" en la barra de busqueda le daras click y le daras a instalar.

Ahora ya se puede usar el programa correctamente, pero no tenemos ningun archivo creado, asi que deberemos crearlo, pero eso lo veremos en la siguiente parte.
